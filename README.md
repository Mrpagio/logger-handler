# Logger Handler

Questo repository contiene la documentazione della libreria in Italiano e in Inglese.

- README (Italiano): `README_IT.md`
- README (English): `README_EN.md`

Comandi rapidi:

```bash
# Eseguire i test (direttamente)
go test -v ./...

# Eseguire i test (script che salva output in test/output)
bash test/run_all_tests.sh

# Eseguire l'esempio principale
go run ./example

# Eseguire l'esempio meter == nil
go run ./example/meter_nil
```

Per i dettagli e la documentazione completa, aprire il file `README_IT.md` o `README_EN.md`.
